Creating an order

Deluxe Avails application provides the facility to create orders for titles. To create an order, you must provide the information for fields displayed on the ORDER MANAGEMENT page.

 

To create an order:

1.    On the Deluxe Avails application home page, click the ORDER MANAGEMENT tab.

The FILTERS page is displayed.

2.    On the FILTERS page, click CREATE ORDER.

The ORDER - CREATE ORDER page is displayed.

3.    On the ORDER - CREATE ORDER page, provide the field values shown in the following table, and then click SAVE.

An order is created successfully.

Critical columns

Following are the critical columns in the Order screen.

Field Name

Description
Order Status
  • New: The order status is set to New when a new title (Territory/Language/Edit/Language type) is identified and inserted in the Deluxe Avails system. Only the orders with the status New appears in the Announcement Analysis Reports.
  • Request received: The order status is set to Request received when the MR-ID inserted against a new record is not available in the pipeline orders.
  • In Progress: The order status is set to In Progress when the details of the MR-ID such as PO and HAL-ID exists in the pipeline report. When you insert the value for MR_ID in the application, the corresponding values of HAL-ID and PO for this record is updated automatically.
  • Complete: The order status is set to Complete when in the QC report, an order has an import date.
  • Cancelled: The order status is set to Cancelled when in the Announcement file, a particular combination of the values of all the sales channel (EST, VOD, POEST) and their formats (SD,HD,4K) are either blank or cancelled.
